Command Line Interface

These documents list all commands and their parameters.

They are meant to be complete yet concise references for the commands and their parameters, but not to provide advice on using them. For guides on processing your own data, see Manuals.

They are meant to be browsed freely, not followed from start to finish. For step-by-step tutorials, see Tutorials.

Note

All subcommands, arguments, and options listed in these documents can be printed, along with a short explanation of each, on the command line by running seismic [command(s)] --help. For example, seismic --help prints all options and subcommands for the command seismic, seismic graph --help prints all options and subcommands for the command seismic graph, and seismic graph seqbar --help prints all options and subcommands for the command seismic graph seqbar.

seismic all

Run ‘align’, ‘relate’, ‘mask’, (optionally) ‘cluster’, ‘table’, (optionally) ‘fold’, and (optionally) ‘graph’, in that order.

seismic all [OPTIONS] FASTA [INPUT_PATH]...

Enable demultiplexing

-x, --fastqx <fastqx>

--parallel-demultiplexing <parallel_demultiplexing>

Whether to run demultiplexing at maximum speed by submitting multithreaded grep functions

--clipped <clipped>

Designates the amount of clipped patterns to search for in the sample, will raise compution time

--mismatch-tolerence <mismatch_tolerence>

Designates the allowable amount of mismatches allowed in a string and still be considered a valid pattern find. will increase non-parallel computation at a factorial rate. use caution going above 2 mismatches. does not apply to clipped sequences.

--index-tolerance <index_tolerance>

Designates the allowable amount of distance you allow the pattern to be found in a read from the reference index

--demulti-overwrite <demulti_overwrite>

-X, --dmfastqx <dmfastqx>
